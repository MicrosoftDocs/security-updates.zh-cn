Microsoft 安全通报 925984
=========================

PowerPoint 中的漏洞可能允许远程执行代码
---------------------------------------

发布时间: 2006年9月27日 | 更新时间: 2006年10月10日

Microsoft 已经完成对此漏洞的公众报告的调查。 我们已经发布了可以解决此问题的 [MS06-058](https://technet.microsoft.com/security/bulletin/ms06-058)。 有关此问题的详细信息（包括可用安全更新的下载链接），请查看 [MS06-058](https://technet.microsoft.com/security/bulletin/ms06-058)。解决的漏洞为 PowerPoint 格式错误的记录漏洞 - [CVE-2006-4694](https://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2006-4694)。
